Этот проект-торговая площадка
<---------------------------------------------------->
Главный файл main.py. Из него производится запуск программы.
<---------------------------------------------------->
В папке db лежит база данных. Спомощью программы SQLite можно просматривать содержимое bd.
<---------------------------------------------------->
В папке templates хранятся дизайны страниц
<---------------------------------------------------->
В папку static/uploads сохряняются картинка, которые загружают пользователи
<---------------------------------------------------->
В папке data хранятся таблицы bd
<---------------------------------------------------->
В папке forms хранятся flask-wtf формы
<---------------------------------------------------->
В папке static/CSS хранятся CSS дизыйны
<---------------------------------------------------->
В папке static/img хранятся картинки для загрузки на сайт
